### Choose your Langauge

#### Hello World in C++:

``` cpp 
#include <iostream>
using namespace std;

int main() {
    cout << "Hello World!"; 
    return 0;
}
```

#### Hello World in Java:

``` java 
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello World!");
    }
}

```


#### Hello World in Python:

``` py
def func():
    print("Hello World!);
```
